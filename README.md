# 开场白生成器

SillyTavern 扩展插件 - 使用 AI 为角色卡生成开场白。

## 功能

- **角色信息读取**: 自动读取角色描述、性格、场景、当前开场白
- **世界书条目选择**: 支持选择角色关联的世界书条目作为参考
- **预设条目选择**: 支持选择预设中的条目参与生成
- **自定义需求**: 可输入开场白风格需求（如神秘感、第一人称等）
- **独立API配置**: 支持 OpenAI / Gemini 格式的 API

## 使用方法

1. 点击扩展菜单中的「开场白生成器」打开弹窗
2. 选择要包含的角色信息（描述/性格/场景/开场白）
3. 点击「选择条目」选择世界书和预设条目（可选）
4. 在需求框中输入想要的开场白风格
5. 点击「生成开场白」等待 AI 生成
6. 预览生成结果，满意后点击「应用」

## API 配置

在「API」标签页中配置：
- API 类型（OpenAI 格式 / Gemini 格式）
- API URL
- API Key
- 模型选择

点击「获取」可自动获取可用模型列表。
